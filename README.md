Python ETL Project using Pandas
------------------------------------------
Project Overview
------------------------------------------

-This project demonstrates a basic ETL (Extract, Transform, Load) pipeline built using Python and Pandas.

-The goal of this project is to read raw sales data from a CSV file, clean and transform it, and generate a processed dataset that is ready for analysis.

-This project is designed as the foundation step in a Data Engineering learning path before moving to PySpark, Databricks, and cloud-based ETL pipelines.


Tech Stack
------------------------------------------------
Python
Pandas
CSV file system
Git & GitHub
